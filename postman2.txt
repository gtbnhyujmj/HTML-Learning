📦 三、其他常用功能
功能	說明
Collection	儲存一系列 API 請求（可分享、版本管理）
Environment	可設定變數（例如 BASE_URL、TOKEN）
Tests	在回應後自動執行 JavaScript 做驗證
Pre-request Script	在發送前執行的 JS 程式碼，例如自動加時間戳
