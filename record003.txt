<ul> 是「無序清單（unordered list）」，用來列出沒有順序的項目，每個項目前會顯示一個 圓點 (•) 或其他符號，不是數字編號。

<ol> → 有序清單（ordered list），會顯示 1、2、3… 這種 編號。

<li>	清單項目	搭配 <ul> 或 <ol> 使用	每一項的內容

✅ <ul> 裡面只能包含 <li>
✅ <li> 可以內嵌 <ul> 或 <ol>，這樣就形成「巢狀清單」或「小項目」

✅ Banner 常見做法之一：用 <ul> + <li> 來排圖片輪播或圖文清單
<ul> 本身是「可排列的區塊容器」
<li> 是「可以放一張圖片或一段文字的項目」
它的結構清晰、容易加 CSS 做動畫、也方便套用 JS 做輪播

#menu li {
  /* CSS 規則 */
}

✅ 這句話的意思是：
所有「在 id 為 menu 裡面」的 <li> 標籤，都會套用這些 CSS 規則。
